Detailed Description

KanbanPage.

Repräsentiert eine interaktive Kanban-Seite zur Verwaltung von Spalten und Issues. Unterstützt Drag&Drop, automatisches Scrollen, Spalten-/Issue-Erstellung und -Aktualisierung.
